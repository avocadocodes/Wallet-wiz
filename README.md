# Wallet App

A wallet application designed to manage your digital transactions with ease. The frontend is deployed on Vercel, and the backend is hosted on Render.

## Table of Contents

- [Features](#features)
- [Tech Stack](#tech-stack)
- [Installation](#installation)
- [Usage](#usage)
## Features

- User authentication and authorization
- Manage and track expenses and income
- Transaction history and reports
- Responsive design with Tailwind CSS

## Tech Stack

**Frontend:**

- React.js + Vite
- Tailwind CSS

**Backend:**

- Node.js
- Express
- MongoDB 

## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/avocadocodes/Wallet-wiz.git
    cd Wallet-wiz
    ```

2. Install dependencies for the frontend:
    ```sh
    cd Wallet-wiz
    npm install
    ```

3. Install dependencies for the backend:
    ```sh
    cd ../server
    npm install
    ```

## Usage

1. Start the backend server:
    ```sh
    cd backend
    npm run start
    ```

2. Start the frontend development server:
    ```sh
    cd ../frontend
    npm run dev
    ```

3. Open your browser and navigate to `http://localhost:5173` to see the application in action.
